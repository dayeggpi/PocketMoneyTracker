# PocketMoneyTracker
Track the pocket money you give to your kids.

## PHP install
1. Copy content of php folder into your root php server
2. Go to index.php
3. ...
4. Profit

## Python install
1. Copy content of python folder into your computer
2. Run `pip install -r requirements.txt`
3. Run `python main.py`
4. ...
5.  Profit

Alternatively, instead of point 3, you can compile to an .exe file using command:<br/>
`pip install pyinstaller`<br/>
followed by <br/>
`pyinstaller --onefile --windowed --name "PocketMoneyTracker" main.py` and run this exe file (in `dist` folder taht will be generated). Or, download an already compiled exe file from this repo.

## data.json
The same data.json file is usable in both python and php config (ie. you can copy it from one to the other construct and it will work).

## How to use : 
1. add kids
2. adjust the allocation as per your needs
3. click on one kids's pocket money
4. add money for a chosen period.
5. adjust as needed
6. see evolution
<br/><br/>
Spent = given to the kids so that they can spend it.<br/>
Given = given by your kids (to charity for example). <br/>
Saved = what you keep aside for your kids, they can use part/all of it in the future.
